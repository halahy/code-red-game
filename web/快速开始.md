# 快速开始

## 方法1：直接打开（最简单）
1. 双击 `index.html` 文件
2. 游戏会在浏览器中打开

## 方法2：使用本地服务器（推荐）
这样可以避免一些浏览器的安全限制。

### Python
```bash
cd web
python -m http.server 8000
```
然后访问：http://localhost:8000

### Node.js
```bash
cd web
npx http-server
```
然后访问显示的地址（通常是 http://localhost:8080）

## 游戏操作
- **WASD** 或 **方向键**: 移动
- **Shift**: 冲刺
- **空格**: 闪避
- **鼠标移动**: 瞄准
- **鼠标左键**: 射击
- **R**: 换弹
- **Q**: 切换武器
- **1/2**: 选择武器槽位
- **ESC**: 暂停/继续

## 注意事项
- 存档保存在浏览器的localStorage中
- 清除浏览器数据会删除存档
- 建议使用Chrome或Edge浏览器以获得最佳体验
