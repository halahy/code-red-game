# 赤焰行动 - Web版

这是《赤焰行动》的Web版本，使用HTML5 Canvas和JavaScript开发。

## 运行方法

### 电脑端

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用本地服务器运行（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```
   然后在浏览器中访问 `http://localhost:8000`

### 手机端

1. **确保手机和电脑在同一WiFi网络**
2. **在电脑上启动服务器**（同上）
3. **查看电脑IP地址**：
   - Windows: `ipconfig` 查看IPv4地址
   - Mac/Linux: `ifconfig` 或 `ip addr`
4. **在手机浏览器中输入**：`http://你的IP地址:8000`
   - 例如：`http://192.168.1.100:8000`

详细说明请查看 `手机端使用说明.md`

## 游戏操作

### 电脑端
- **WASD** 或 **方向键**: 移动
- **Shift**: 冲刺
- **空格**: 闪避
- **鼠标移动**: 瞄准
- **鼠标左键**: 射击
- **R**: 换弹
- **Q**: 切换武器
- **1/2**: 选择武器槽位
- **ESC**: 暂停/继续

### 手机端
- **虚拟摇杆**（左下角）: 移动
- **触摸画布滑动**: 瞄准
- **射击按钮**（红色大按钮）: 按住射击
- **换弹按钮**: 点击换弹
- **闪避按钮**: 点击闪避
- **切换武器按钮**: 点击切换武器

## 功能特性

- ✅ 完整的游戏机制（移动、射击、闪避）
- ✅ 多种武器系统
- ✅ 波次和关卡系统

- ✅ **移动端支持**（触摸控制、虚拟摇杆、响应式设计）
