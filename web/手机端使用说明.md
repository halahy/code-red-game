## 💻 本地测试（仅用于开发）

如果你想在本地测试，可以使用以下方法：

### 方法1：通过浏览器访问（需要同一WiFi）

1. **确保手机和电脑在同一WiFi网络**

2. **在电脑上启动本地服务器**：
   ```bash
   cd web
   python -m http.server 8000
   ```

3. **查看电脑的IP地址**：
   - Windows: `ipconfig` 查看IPv4地址
   - Mac/Linux: `ifconfig` 或 `ip addr`

4. **在手机浏览器中输入**：
   ```
   http://你的IP地址:8000
   ```

## 移动端操作说明

### 虚拟摇杆（左下角）
- **拖动摇杆**：控制角色移动
- 支持8方向移动

### 按钮（右下角）
- **射击按钮**（红色大按钮）：按住射击
- **换弹按钮**：点击换弹
- **闪避按钮**：点击闪避
- **切换武器按钮**：点击切换武器

### 触摸瞄准
- **在画布上滑动**：控制瞄准方向
- 角色会朝向触摸点瞄准

## 优化建议

1. **横屏游戏**：建议将手机横屏以获得更好的游戏体验
2. **性能**：如果游戏卡顿，可以关闭其他应用
3. **网络**：使用WiFi连接以获得更好的体验

## 常见问题

**Q: 游戏无法加载？**
A: 确保手机和电脑在同一WiFi网络，并且防火墙允许8000端口

**Q: 触摸控制不灵敏？**
A: 尝试横屏游戏，或者调整浏览器缩放

**Q: 游戏画面太小？**
A: 横屏游戏可以获得更大的显示区域

**Q: 如何全屏游戏？**
A: 在浏览器菜单中选择"添加到主屏幕"，然后从主屏幕打开
