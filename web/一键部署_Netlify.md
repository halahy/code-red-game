# 最简单部署方法 - Netlify（3步完成）

## 🚀 超简单3步部署

### 第1步：访问网站
打开 https://www.netlify.com

### 第2步：注册/登录
- 点击右上角 "Sign up"
- 可以用GitHub账号、Google账号或邮箱注册
- 完全免费

### 第3步：拖拽部署
1. 登录后，你会看到一个大的拖拽区域
2. 直接将整个 `web` 文件夹拖进去
3. 等待几秒钟，网站就部署好了！

## 📱 获得网址

部署完成后，你会得到一个网址，例如：
```
https://your-game-name-123.netlify.app
```

## 🎮 分享给朋友

把这个网址发给朋友，他们：
- 在手机上打开浏览器
- 输入网址
- 直接就能玩！

## ✨ 可选：自定义网址

1. 点击部署好的网站
2. 点击 "Site settings"
3. 点击 "Change site name"
4. 输入你想要的名称（如：`code-red-game`）
5. 新网址：`https://code-red-game.netlify.app`

## 🆘 遇到问题？

**Q: 拖拽后没有反应？**
A: 确保拖的是 `web` 文件夹，不是其他文件夹

**Q: 部署失败？**
A: 检查 `web` 文件夹里是否有 `index.html` 文件

**Q: 网址打不开？**
A: 等待1-2分钟，有时候需要一点时间生效
