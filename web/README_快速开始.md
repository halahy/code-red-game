# 🎮 快速开始 - 5分钟部署游戏

## 🚀 最简单的方法：Netlify

### 3步完成部署：

1. **访问** https://www.netlify.com
2. **注册**（免费，可用GitHub/Google账号）
3. **拖拽** `web` 文件夹到页面

**完成！** 你会得到一个网址，例如：`https://your-game-123.netlify.app`

### 分享给朋友：

直接把网址发给他们，他们在手机上打开浏览器就能玩！

---

## 📱 手机端操作

- **左下角摇杆**：移动
- **右下角按钮**：射击、换弹、闪避、切换武器
- **画布上滑动**：瞄准

建议横屏游戏！
