# 游戏部署指南 - 一键分享给朋友

## 方法1：GitHub Pages（推荐，完全免费）

### 步骤：

1. **创建GitHub账号**（如果没有）
   - 访问 https://github.com
   - 注册账号

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名：`code-red-game`（或任意名字）
   - 选择 Public（公开）
   - 点击 "Create repository"

3. **上传文件**
   - 方法A：使用GitHub网页上传
     - 点击 "uploading an existing file"
     - 将 `web` 文件夹里的所有文件拖进去
     - 点击 "Commit changes"
   
   - 方法B：使用Git命令（如果已安装Git）
     ```bash
     cd web
     git init
     git add .
     git commit -m "Initial commit"
     git branch -M main
     git remote add origin https://github.com/你的用户名/code-red-game.git
     git push -u origin main
     ```

4. **启用GitHub Pages**
   - 在仓库页面，点击 "Settings"
   - 左侧菜单找到 "Pages"
   - Source 选择 "main" 分支，文件夹选择 "/ (root)"
   - 点击 "Save"
   - 等待1-2分钟，会显示你的网址：`https://你的用户名.github.io/code-red-game/`

5. **分享链接**
   - 直接把这个网址发给朋友，他们就可以在手机上打开玩了！

---

## 方法2：Netlify（最简单，拖拽即可）

### 步骤：

1. **访问 Netlify**
   - 打开 https://www.netlify.com
   - 点击 "Sign up" 注册（可以用GitHub账号登录）

2. **部署网站**
   - 登录后，将 `web` 文件夹直接拖到 Netlify 的部署区域
   - 或者点击 "Add new site" → "Deploy manually"
   - 上传 `web` 文件夹

3. **获取链接**
   - 部署完成后，会得到一个随机网址，如：`https://random-name-123.netlify.app`
   - 可以点击 "Site settings" → "Change site name" 修改为自定义名称

4. **分享链接**
   - 把网址发给朋友即可！

---

## 方法3：Vercel（适合开发者）

### 步骤：

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 用GitHub账号登录

2. **导入项目**
   - 点击 "Add New Project"
   - 如果代码在GitHub，直接选择仓库
   - 或者上传 `web` 文件夹

3. **部署**
   - 点击 "Deploy"
   - 几秒钟后获得网址
